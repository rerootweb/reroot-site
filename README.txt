ReRoot Space Website (Vercel quick deploy)

1) Upload this folder to a new GitHub repo called `reroot-site` (Add file → Upload files).
2) On vercel.com: New Project → Import Repo.
   - Framework: Vite
   - Build: npm run build
   - Output: dist
3) Done. You can edit text/links in src/App.jsx. Images live in /public.
